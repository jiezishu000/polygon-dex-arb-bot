# Polygon DEX Arbitrage Bot

Real-time cross-DEX arbitrage scanner for QuickSwap & SushiSwap on Polygon.
Monitors MATIC/USDC, MATIC/USDT, WETH/USDC, WETH/USDT pairs for price spreads.

## Quick Start
```bash
pip install web3 requests
python polygon_swapper.py scan      # Scan for opportunities
python polygon_swapper.py monitor   # 24/7 monitoring
python polygon_swapper.py status    # Wallet & network status
```
